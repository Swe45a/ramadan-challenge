#  Ramadan Challenge Project

A modern web application designed to inspire and track daily good deeds during the holy month of Ramadan.  
The platform encourages consistency, motivation, and community engagement through structured daily challenges.

---

##  Features

-  Daily Ramadan Challenges  
-  Streak Tracking System  
-  Motivational Messages  
-  Progress Dashboard  
-  Leaderboard & Community Impact  
-  AI Chat Support  
-  Local Storage + Supabase Integration  

---

##  Tech Stack

**Frontend:**
- React + TypeScript  
- Vite  
- Tailwind CSS  

**Backend:**
- Node.js  
- Express  

**Database & Services:**
- Supabase
